## Mako code

## !bbox environments lead to minipage, which cannot contain figures.
## Solution: use !bbox except for latex.
## Code: ${bbox()} and ${ebox()} instead of !bbox and !ebox

<%
def boxenvir(tp='b'):
    if FORMAT not in ('latex', 'pdflatex'):
        return '!%sbox' % tp
    else:
        return ''

def bbox():
    return boxenvir('b')

def ebox():
    return boxenvir('e')
%>
